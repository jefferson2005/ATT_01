Automatizando o desenvolvimento

O que é

O Git é um sistema de controle de versão distribuído amplamente utilizado no desenvolvimento de software, permitindo que equipes gerenciem e rastreiem alterações em arquivos de código-fonte ao longo do tempo. Diferente de sistemas de armazenamento convencionais, o Git não apenas salva versões atualizadas dos arquivos, mas mantém um histórico completo e estruturado de todas as modificações realizadas, criando uma linha do tempo detalhada do projeto. Cada vez que um desenvolvedor faz uma alteração no código e a registra no Git (por meio de um commit), o sistema não só armazena o estado atual dos arquivos,mas também guarda metadados importantes, como quem fez a mudança, quando foi realizada e uma mensagem descritiva explicando o propósito da modificação. Isso permite que, se necessário, a equipe possa revisitar versões anteriores comparar alterações entre outras funções.

O que é

O GitHub é uma plataforma online que permite armazenar, compartilhar e colaborar em projetos de programação usando o Git. Ele funciona como uma "nuvem" onde você pode salvar o código do seu projeto e acompanhar todas as mudanças feitas por você ou por sua equipe. Além de versionar o código, o GitHub facilita o trabalho em grupo, permitindo que várias pessoas editem o mesmo projeto sem perder controle das alterações. Também é muito usado para mostrar portfólios de desenvolvedores, contribuir com projetos open source e automatizar processos de desenvolvimento.

Algumas funções e ferramentas

O Git é um sistema de controle de versões que ajuda a organizar e acompanhar as mudanças em projetos. Ele possui algumas ferramentas principais para facilitar seu uso:
  • Git GUI: É a versão com interface gráfica. Ideal para iniciantes, permite fazer commits, merges e outras ações com cliques, sem usar comandos.
  • Git CMD: É parecido com o prompt de comando do Windows. Permite usar comandos Git, mas com menos recursos que o Git Bash.
  • Git Bash: É o terminal mais completo para usar o Git. Suporta todos os comandos e é muito usado por desenvolvedores. Com ele, você pode criar branches, fazer commits, enviar projetos para o GitHub e muito mais.

Como utilizar os comandos básicos Git

Verificando a versão

git --version Verifica se o Git está instalado e mostra a versão.

Criando o repositório

git init Inicia um novo repositório Git na pasta atual.

Adicionado seu projeto

git add Adiciona todas as alterações feitas nos arquivos para serem salvas no próximo commit.
